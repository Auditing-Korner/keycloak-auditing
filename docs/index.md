# Keycloak Auditor

Keycloak Auditor is a modular security auditing framework for Keycloak. It integrates Nuclei workflows/templates and AI-guided checks to enumerate, audit, scan, safely validate, and report on Keycloak deployments.

- Enumeration of realms, clients, roles, groups, IDPs, flows
- Configuration audit against best practices
- Vulnerability scanning using local Nuclei templates/workflows
- Safe, non-destructive exploitation checks
- Consolidated Markdown and JSON reporting
